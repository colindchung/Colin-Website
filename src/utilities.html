BiLLy Receipt Management

Visit project repository

Created: May 2019

Complexity:

What's The Stack?

I attended a hackathon at Ryerson University with 3 of my friends. We used Google Vision AI for image processing to create a receipt management system. It is a web application which allows the user to upload images of receipts or bills and store transaction information in the database. The backend was created using Node.js with data resting in MongoDB and accessed using API endpoints built using Express.js. The frontend was built with React because of the table preview in the user interface which needs to constantly re-render.

This project is still a work in progress. Currently, we are optimizing the algorithms which take the Google Vision AI results and determine the vendor, date, transaction type and total payment. We would like to add more columns to the table such as specific item information and item categories. We are going to give functionality to the Stats and History tab which can allow a user to download a spreadsheet copy of their transaction history and filter using different criteria. Lastly, we will need to improve the login process and incorporate OAuth.

What's The Stack?

I made a custom library, primarily for data science functions. Most of the functions I made are not available in the standard math library, or that are not available in the format that I like to use. I import my library class in other C# projects with data analysis for smooth programming. Some interesting methods I made are:

  • Convert each entry in a list to integer, double
  • Mean, median, mode averages
  • Standard deviation
  • Permutations, factorials
  • Matrix algebra
  • Vector algebra
  • Area, volume
