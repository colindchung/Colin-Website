Hat Catalog

Visit project repository

Visit the website!

Created: April 2019

Complexity:

Programming Languages and Techniques Used:

  • HTML
  • CSS
  • JavaScript
  • jQuery

I love collecting hats, so I made a website to catalog them. I used jQuery to give it a dynamic flare, with animations on clicks and hovers. When you hover over a hat, the image enlarges and you can click a category to show or hide it.

On loading, the categories are hidden.

Clicking a category reveals the hats.

And hovering over it enlarges it!

I created a website about the history of my favourite sports team, the San Antonio Spurs. This project helped me learn fundamental HTML and CSS structure and styling basics. I have a <div> sectioned off for each significant era in Spurs history. The CSS integration is primarily class and id styling, formatting the website to be more visually appealing.
